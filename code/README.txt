
* Les trois dossiers représentent les trois parties du cours:

- Bandit : implémentation des algorithmes Greedy, Greedy Epsilon, Softmax, UCB
- MDP : le modèle MDP appliqué à un labyrinthe avec les algorithmes TD, QLearning, MonteCarlo.
- Go : implémentation du moteur de Go avec Monte Carlo Search

* Chaque dossier a ses spécificités (un fichier README.txt se trouve dans chaque dossier), mais les trois requièrent Python 2.7:
 (http://www.python.org/download/releases/2.7/)
